# 2D Objects — 2D オブジェクト

`2d/` 以下のオブジェクト。`t_vec2` と `t_2d_transform` を使い、2D 空間で描画される。

## 3D オブジェクトとの違い

| | 3D オブジェクト | 2D オブジェクト |
|---|---|---|
| 座標 | `t_vec3` | `t_vec2` |
| トランスフォーム | `t_transform` | `t_2d_transform` |
| 変換関数 | `gl_rotate`, `gl_translate`, `gl_scale` | `gl_2d_rotate`, `gl_2d_translate`, `gl_2d_scale` |

---

## 2D Line

### 型定義

```c
typedef struct s_2d_line {
    t_object_type  type;       // GL_TYPE_2D_LINE
    char           label[32];
    t_vec2         start;
    t_vec2         end;
    t_color        color;
    t_2d_transform transform;
} t_2d_line;
```

### 関数

| シグネチャ | 説明 |
|---|---|
| `t_2d_line gl_new_2d_line(t_vec2 start, t_vec2 end)` | 2D 線を作成 |
| `t_2d_line gl_new_2d_line_with_label(t_vec2 start, t_vec2 end, const char *label)` | ラベル付きで作成 |
| `t_2d_line *gl_alloc_2d_line(t_vec2 start, t_vec2 end)` | ヒープに確保 |
| `t_2d_line *gl_alloc_2d_line_with_label(t_vec2 start, t_vec2 end, const char *label)` | ラベル付きでヒープに確保 |
| `void gl_free_2d_line(t_2d_line *line)` | 解放 |
| `bool gl_label_2d_line(t_2d_line *line, const char *label)` | ラベルを設定 |
| `void gl_color_2d_line(t_2d_line *line, t_color color)` | 色を設定 |
| `void gl_draw_2d_line(t_2d_line *line)` | 描画 |

### 使用例

```c
t_2d_line line = gl_new_2d_line(gl_vec2(0, 0), gl_vec2(100, 100));
gl_color_2d_line(&line, gl_red());
gl_draw_2d_line(&line);
```

## 2D Rect

### 型定義

```c
typedef struct s_2d_rect {
    t_object_type  type;          // GL_TYPE_2D_RECT
    char           label[32];
    t_vec2         origin;
    t_vec2         size;
    t_color        color;
    bool           has_outline;
    t_color        outline_color;
    t_2d_transform transform;
} t_2d_rect;
```

### 関数

| シグネチャ | 説明 |
|---|---|
| `t_2d_rect gl_new_2d_rect(t_vec2 origin, t_vec2 size)` | 2D 矩形を作成 |
| `t_2d_rect gl_new_2d_rect_with_label(t_vec2 origin, t_vec2 size, const char *label)` | ラベル付きで作成 |
| `t_2d_rect *gl_alloc_2d_rect(t_vec2 origin, t_vec2 size)` | ヒープに確保 |
| `t_2d_rect *gl_alloc_2d_rect_with_label(t_vec2 origin, t_vec2 size, const char *label)` | ラベル付きでヒープに確保 |
| `void gl_free_2d_rect(t_2d_rect *rect)` | 解放 |
| `bool gl_label_2d_rect(t_2d_rect *rect, const char *label)` | ラベルを設定 |
| `void gl_color_2d_rect(t_2d_rect *rect, t_color color)` | 色を設定 |
| `void gl_outline_2d_rect(t_2d_rect *rect, t_color color)` | 枠線を設定 |
| `void gl_draw_2d_rect(t_2d_rect *rect)` | 描画 |

### 使用例

```c
t_2d_rect rect = gl_new_2d_rect(gl_vec2(10, 10), gl_vec2(200, 100));
gl_color_2d_rect(&rect, gl_color(0.2, 0.6, 1.0));
gl_outline_2d_rect(&rect, gl_black());
gl_2d_rotate(&rect.transform, 15.0);
gl_draw_2d_rect(&rect);
```
